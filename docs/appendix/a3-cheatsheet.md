## 这一页该怎么用

先说清楚这页是干嘛的，免得你误会。它**不是用来第一次学知识的**——公式背后的"为什么"，都在第 1 章那几节里讲透了，这里只把结论拎出来排成表。换句话说，它更像考前你贴在桌角的那张小抄：每个公式你都已经懂了，只是临场怕记混，扫一眼就能确认。

所以正确的用法是：第一次遇到某个概念，去对应的小节老老实实学一遍；学完之后，再回到这页，把它当成"知识的索引和提醒"。如果你扫到某一行发现"咦，这是什么"，那就是个信号——说明那块还没真懂，顺着链接回去补就好。

## 线性代数速查

下面这几条是你在神经网络、回归、降维里会反复用到的。记不住公式不要紧，但每一条"在解决什么问题"要心里有数。

| 名称 | 公式 / 记号 | 一句话提醒 |
| --- | --- | --- |
| 矩阵乘法 | $C_{ij}=\sum_k A_{ik}B_{kj}$ | $A$ 的列数要等于 $B$ 的行数；本质是对列做加权求和 |
| 矩阵乘向量 | $\mathbf{y}=A\mathbf{x}$ | 神经网络一层的核心动作；代码里用 `@` 不是 `*` |
| 转置 | $(A^\top)_{ij}=A_{ji}$ | 行列互换，常用来"对齐形状"好让两矩阵能乘 |
| 逆 | $A^{-1}A=I$ | "撤销操作"；解方程优先用 `solve`，别滥用求逆 |
| 范数（L2） | $\lVert\mathbf{x}\rVert_2=\sqrt{\sum_i x_i^2}$ | 度量向量长度 / 两点距离；正则化里也是它 |
| 点积 | $\mathbf{a}\cdot\mathbf{b}=\sum_i a_ib_i$ | 衡量两向量"有多同向"，注意力打分就是它 |
| 特征值 | $A\mathbf{v}=\lambda\mathbf{v}$ | 变换里"方向不变只被拉伸"的那些方向 |
| SVD | $A=U\Sigma V^\top$ | 任意矩阵都能拆；PCA 降维靠的就是它 |

更细的推导和几何直觉，回 [3.1 线性代数](../ch1-intro/1-3-1-linear-algebra.md) 看。

## 微积分与梯度速查

训练模型说到底就是"沿梯度反方向下山"，所以这几条是引擎级别的，必须熟。

| 名称 | 公式 | 一句话提醒 |
| --- | --- | --- |
| 导数 | $f'(x)=\lim_{h\to0}\dfrac{f(x+h)-f(x)}{h}$ | 曲线某点的斜率，变化快慢 |
| 梯度 | $\nabla f=\big(\frac{\partial f}{\partial x_1},\dots,\frac{\partial f}{\partial x_n}\big)$ | 指向"上升最快"的方向，反方向才是下山 |
| 链式法则 | $\dfrac{dy}{dx}=\dfrac{dy}{du}\cdot\dfrac{du}{dx}$ | 反向传播的命根子，复合函数逐层相乘 |
| 梯度下降 | $\theta\leftarrow\theta-\eta\nabla f$ | $\eta$ 是学习率，太大震荡、太小爬得慢 |
| Sigmoid 导数 | $\sigma'=\sigma(1-\sigma)$ | 逻辑回归、激活函数里直接套用 |

铺垫和手推过程在 [3.2 微积分与梯度](../ch1-intro/1-3-2-calculus-gradients.md)。

## 概率与统计速查

机器学习的损失函数、评估指标，背后几乎都站着概率。这几条出现频率最高。

| 名称 | 公式 | 一句话提醒 |
| --- | --- | --- |
| 期望 | $\mathbb{E}[X]=\sum_x x\,p(x)$ | 取值按概率加权的"平均" |
| 方差 | $\mathrm{Var}(X)=\mathbb{E}[(X-\mu)^2]$ | 数据有多分散 |
| 贝叶斯 | $P(A\mid B)=\dfrac{P(B\mid A)P(A)}{P(B)}$ | 由"已知结果"反推"原因"的概率 |
| Softmax | $\mathrm{softmax}(z)_i=\dfrac{e^{z_i}}{\sum_j e^{z_j}}$ | 把一组分数变成和为 1 的概率；先减最大值防溢出 |
| 交叉熵 | $L=-\sum_i y_i\log\hat{y}_i$ | 分类最常用的损失，预测越偏惩罚越重 |
| 均方误差 | $L=\frac{1}{n}\sum_i(\hat{y}_i-y_i)^2$ | 回归最常用的损失 |

想把这些讲透，去 [3.3 概率与统计](../ch1-intro/1-3-3-probability-statistics.md)；优化相关的凸性结论在 [3.4 凸优化基础](../ch1-intro/1-3-4-convex-optimization.md)。

## 中英术语对照表

这张表解决一个很实际的问题：赛题、论文、库文档基本都是英文，你脑子里的中文概念得能瞬间对上英文词，反应才跟得上。挑的都是全书高频、最容易在英文里"愣住"的那些。

| 中文 | English | 一句话含义 |
| --- | --- | --- |
| 过拟合 | overfitting | 把训练集背得太死，遇到新数据就崩 |
| 欠拟合 | underfitting | 模型太简单，连训练集都学不好 |
| 梯度 | gradient | 损失对参数的偏导，指向上升最快方向 |
| 反向传播 | backpropagation | 用链式法则把梯度从输出逐层传回参数 |
| 学习率 | learning rate | 每步迈多大，最关键的超参数之一 |
| 损失函数 | loss function | 衡量"预测离真相有多远"的那把尺子 |
| 嵌入 | embedding | 把离散对象（词、物品）映成稠密向量 |
| 正则化 | regularization | 给模型加约束，防止它过拟合 |
| 特征 | feature | 描述一个样本的一个属性 / 维度 |
| 张量 | tensor | 多维数组，深度学习的数据基本单位 |
| 批量 | batch | 一次喂给模型的一小撮样本 |
| 轮次 | epoch | 把整个训练集完整过一遍 |
| 超参数 | hyperparameter | 训练前手动设、不由训练学出来的参数 |
| 注意力 | attention | 让模型按相关性给不同位置分配权重 |
| 微调 | fine-tuning | 在预训练模型上接着小步训练适配新任务 |
| 偏差 / 方差 | bias / variance | 模型"系统性偏错"与"对数据太敏感"的两难 |

看到陌生术语想知道它"怎么用、考什么"，可以从 [考纲总览与学习路线](../ch1-intro/1-2-syllabus-roadmap.md) 顺藤摸瓜找到对应章节。

## 容易踩的坑

- **把速查当学习**：直接背表格而跳过原节，等于背了答案没懂题——考场上稍微换个壳子就抓瞎。表是用来"提醒"的，不是用来"代替理解"的。
- **`@` 和 `*` 记混**：矩阵乘法是 `@`，逐元素乘是 `*`，这俩在公式里都长得像"乘"，但代码结果天差地别。
- **梯度方向记反**：梯度指"上升"最快，下山要用**负**梯度，更新公式里那个减号千万别丢。
- **Softmax 忘了数值稳定**：直接对大分数取指数会溢出，记得先减去最大值（结果不变）。

## 小结

这一页只有一个用法：**先去对应小节把每个公式学懂，再回来把它当索引和考前小抄。** 公式速查管"算什么"，术语表管"英文怎么说"，两张表配合着用，第 1 章的数学底子就能随调随取。想系统补推导，回 [3.1 线性代数](../ch1-intro/1-3-1-linear-algebra.md)、[3.2 微积分与梯度](../ch1-intro/1-3-2-calculus-gradients.md) 和 [3.3 概率与统计](../ch1-intro/1-3-3-probability-statistics.md)；想要一本权威参考书放手边，推荐 [*Mathematics for Machine Learning*](https://mml-book.github.io/)（免费在线，覆盖了本页全部公式）。
