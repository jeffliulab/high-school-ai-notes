## 为什么会有"没讲的东西"

你可能会奇怪：一本备赛教程，怎么还专门列一页"我们不讲什么"？

原因很实在。AI 是一片大得吓人的领域，强化学习、图神经网络、传统计算机视觉、知识图谱……每一个拎出来都能写一整本书。但**我们的目标很明确：帮你高效地通过 USAAIO，进而冲击 IOAI / IAIO。** 既然如此，凡是不在这几个比赛考纲（syllabus）里的内容，学了对拿分没有直接帮助，反而会分散你本就紧张的备赛时间。

所以这一页不是"这些东西不重要"，而是"**这些东西重要，但不在这场考试的射程内**"。我们把它们诚实地标出来，再给你一个入门的指路牌——这样你既不会在备赛期走神，又能在比赛之后知道往哪儿继续探索。判断"某主题到底考不考"的最终依据，永远是 [2. 考纲总览与学习路线](../ch1-intro/1-2-syllabus-roadmap.md) 里贴出的官方考纲，本页只是对它的补充说明。

## 我们刻意略过的主题，以及为什么

下面这张表把主要的"未涵盖主题"列在一起。第一列是主题，第二列说清为什么不讲，第三列给一个第一步该看什么的指向。你不用现在就去看这些，扫一眼、心里有数就够了。

| 未涵盖主题 | 为什么不讲 | 想入门可以从这里开始 |
| --- | --- | --- |
| **强化学习（Reinforcement Learning, RL）** | 智能体靠"试错 + 奖励"自己学策略，是个独立的大分支。它不在 USAAIO / IOAI 现行考纲里，且训练通常要环境模拟和大量算力，和比赛"给定数据集做监督学习"的形态对不上。 | David Silver 的 RL 公开课；OpenAI Gymnasium 上手跑一个小例子 |
| **传统（非深度学习）计算机视觉** | 像 SIFT、HOG 这类手工特征，和形态学、霍夫变换等经典算法，在深度学习时代已基本被卷积网络取代。考纲里的 CV 是围绕神经网络展开的（见 [3.1 卷积与池化](../ch2-round1/2-3-1-conv-pooling.md)）。 | OpenCV 官方教程的图像处理章节 |
| **图神经网络（Graph Neural Networks, GNN）** | 专门处理"图结构"数据（社交网络、分子）。概念较新、较专，不在现行考纲内。 | 斯坦福 CS224W；PyTorch Geometric 文档 |
| **概率图模型 / 贝叶斯网络** | 隐马尔可夫模型、贝叶斯网络等是经典的概率建模框架，理论漂亮但比赛几乎不直接考。你需要的概率基础在 [3.3 概率与统计](../ch1-intro/1-3-3-probability-statistics.md) 里已经够用。 | Koller 的《Probabilistic Graphical Models》课程 |
| **知识图谱 / 符号 AI / 专家系统** | 属于"老派"的符号主义路线，和当下以数据驱动的深度学习是两套思路，不在考纲覆盖范围。 | 任意一篇知识图谱综述即可建立概念 |
| **大模型的工程化部署（MLOps、分布式训练框架等）** | 这些是把模型搬上生产环境的工程技能，偏开发运维，与"做题拿分"关系不大。比赛里你只需在单机 / Colab 上把模型跑通。 | 先把 [A4 环境与工具](a4-tools.md) 吃透，工程进阶再另说 |
| **AI 伦理、安全与对齐** | 非常重要的现实议题，但不是 USAAIO 的考点。了解它对你做负责任的研究有益，对刷分无直接帮助。 | 任意一门《AI Safety》入门读物 |

需要强调一句：考纲会逐年微调。**如果某一年官方把上面某个主题（最有可能是强化学习）正式纳入了考纲，那本页就该相应更新，对应内容也会补成正式章节。** 所以临考前，请务必以当年官方 syllabus 为准，别只看这张表。

## 那"考纲里"的主题都在哪

把"不讲的"列清楚，反过来也是想让你看清"**该讲的我们一个没落**"。你现在该投入精力的，是这几条主线：

- 经典机器学习（监督 / 无监督）——见 [Round 1 · 资格赛](../ch2-round1/index.md)
- 深度学习与神经网络基础——见 [2.1 感知机与 MLP](../ch2-round1/2-2-1-perceptron-mlp.md)
- 计算机视觉与自然语言处理（基于深度学习）——见 [Round 2 · 邀请赛](../ch3-round2/index.md)
- Transformer 与预训练大模型——见 [2.2 Transformer 架构](../ch3-round2/3-2-2-transformer.md)、[4.4 预训练大模型](../ch3-round2/3-4-4-pretrained-llms.md)

一句话：**先把考纲内的吃透，本页这些主题，留到比赛之后当兴趣去探索，时间花得才划算。**
