## 先说清楚：这一关到底在考什么

Round 1 是整个比赛的资格赛，也是大多数同学真正动手写代码的起点。它有两个很关键的限制，理解了它们，你才知道该把力气花在哪。

第一，**它只给你 CPU，没有 GPU**。这意味着考场上跑不动那些动辄要训练几个小时的大模型，所以题目考的都是"小而经典"的方法——线性回归、逻辑回归、决策树、一个几层的小神经网络、一个浅层的卷积网络。换句话说，这一关比的不是谁的模型大，而是谁把基础打得更扎实。

第二，**它重在"理解原理"而不是"会用框架"**。正因为如此，本关每一页核心算法我们都坚持一个写法：**先用 NumPy 或 PyTorch 从零手写一遍，再用 scikit-learn 或 torch 的现成接口对照一遍**。从零写，是为了让你真懂它在算什么；库版对照，是为了让你在考场上写得又快又不出错。两样都会，你才算真的掌握了它。

## 本关的四个分类，怎么排顺序

这一关的内容分成四块，我们建议你**基本按顺序往下走**——后面的内容往往要踩着前面的肩膀。下面逐块说一下它们各自管什么。

**第一块，经典机器学习。** 这是不靠神经网络、在 CPU 上就能跑得飞快的一批方法，也是 Round 1 出题最密集的地方。我们从最基础的[监督学习工作流](2-1-1-supervised-workflow.md)讲起，告诉你"训练—验证—测试"这套规矩为什么必须遵守；然后一路学[线性回归](2-1-2-linear-regression.md)、[逻辑回归与分类](2-1-3-logistic-regression.md)、[分类评估指标](2-1-4-classification-metrics.md)、[KNN 与决策树](2-1-5-knn-decision-trees.md)、[集成方法](2-1-6-ensembles.md)、[支持向量机 SVM](2-1-7-svm.md)、[无监督学习](2-1-8-unsupervised.md)，最后用[模型评估与选择](2-1-9-model-evaluation.md)和[进阶特征工程](2-1-10-feature-engineering.md)把整套流程收口。

**第二块，深度学习入门。** 打好经典 ML 的底子，我们就可以迈进神经网络了。从[感知机与 MLP](2-2-1-perceptron-mlp.md)这个最小的神经网络讲起，再把训练它的引擎——[梯度下降与反向传播](2-2-2-backprop.md)——一步步推清楚；接着补上[激活函数与损失函数](2-2-3-activations-losses.md)这两块"零件"，学会用[PyTorch 基础](2-2-4-pytorch-basics.md)把网络真正搭起来，最后用[优化与正则化](2-2-5-optimization-regularization.md)解决"训得动但训不好"的问题。

**第三块，卷积神经网络入门。** 当数据从表格变成图片，普通的 MLP 就不够用了。我们先讲清[卷积与池化](2-3-1-conv-pooling.md)这两个为图像量身定做的运算，再做一次完整的[图像分类实战](2-3-2-image-classification.md)，最后学[迁移学习与预训练编码器](2-3-3-transfer-learning.md)——在 CPU 上不重头训练，而是借别人训练好的模型，这是 Round 1 里又快又有效的一招。

**第四块，Round 1 备考。** 知识学完了，最后要把它们用到真题上。[真题形式与评分](2-4-1-exam-format.md)告诉你考场长什么样、分数怎么给；[模拟与查漏](2-4-2-mock-review.md)则带你做整套模拟、找出自己的薄弱环节，临门一脚把窟窿补上。

## 它和前后关卡怎么衔接

往前看，本关用到的数学和编程工具都在第一章备齐了：算法推导离不开[3.2 微积分与梯度](../ch1-intro/1-3-2-calculus-gradients.md)和[3.1 线性代数](../ch1-intro/1-3-1-linear-algebra.md)，写代码和洗数据则靠[4.2 NumPy 与张量](../ch1-intro/1-4-2-numpy-tensors.md)和[4.3 Pandas 与数据处理](../ch1-intro/1-4-3-pandas-data.md)。如果哪个工具不熟，随时翻回去补。

往后看，Round 1 是地基，[Round 2 · 邀请赛](../ch3-round2/index.md)才是把楼盖高的地方——那里有了 GPU，才谈得上注意力机制、Transformer、目标检测这些"大块头"。可以说，**这一关你练得越扎实，后面就越省力**。打牢基础，我们开始吧。
