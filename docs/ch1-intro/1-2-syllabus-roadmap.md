## 一、IOAI 考纲四大板块

官方把每个知识点标注为 **Theory（懂原理）/ Practice（会用会写）/ Both（两者都要）**。

| 板块 | 代表知识点 | 侧重 |
|---|---|---|
| **1. 基础与经典 ML** | Python、NumPy/Pandas、线性/逻辑回归、决策树、集成、SVM、K-Means、PCA、模型评估 | 多为 Practice / Both |
| **2. 神经网络与深度学习** | 感知机、反向传播、激活/损失、优化器、正则化、注意力、Transformer、微调 | 多为 Both |
| **3. 计算机视觉 CV** | 卷积、图像分类、目标检测、分割、CLIP、GAN、扩散模型 | 多为 Practice |
| **4. NLP 与音频** | 文本分类、BERT、语言建模、预训练大模型、Whisper/HuBERT | 多为 Practice |

!!! warning "范围边界（很重要）"
    考纲遵循"**未列出即排除**"。因此 2026 不考 **RNN/LSTM/GRU、强化学习、时间序列预测**——本站也不讲，原因与选学指引见 [附录 A5](../appendix/a5-excluded.md)。框架方面 **USAAIO 只用 PyTorch**（不用 TensorFlow）。

## 二、知识点 × 四关卡难度矩阵

同一主题在不同关卡是"**深度递增**"，不是换知识点。以"梯度下降"为例：

| 主题 | Round 1 | Round 2 | Camp | Champion |
|---|---|---|---|---|
| 梯度下降 | 会用、知道在干嘛 | 懂动量/Adam/学习率调度 | 诊断不收敛、改 schedule | 真题里为刷分做工程调参 |
| 整体范围 | 经典 ML + DL 入门 + CNN | 全考纲 + Transformer/CV/NLP | 综合实战 + 真题攻坚 | 国际赛真题 + 夺牌策略 |
| 算力 | 仅 CPU | 允许 GPU | — | 国际赛环境 |

> 本站的章节结构正是这条主轴：[第2章 Round 1](../ch2-round1/index.md) → [第3章 Round 2](../ch3-round2/index.md) → [第4章 Camp](../ch4-camp/index.md) → [第5章 Champion](../ch5-champion/index.md)。

## 三、本站统一的「三段式」学习法

每篇知识点笔记都按同一套循序渐进的结构展开，确保你"懂原理、会推导、能落地"：

1. **直觉** —— 用生活化类比建立动机：它解决什么问题？
2. **数学** —— 适度推导关键公式，配 worked example。
3. **代码** —— 从零实现（NumPy/PyTorch）+ 库版（sklearn/torch）对比。
4. 再加 **常见陷阱 → 应用场景 → 练习题 → 小结**。

!!! tip "为什么这样学"
    IOAI 真题要求**既能数学推导、又能 PyTorch 快速实现、且跨领域**。只背 API 会在变体题上翻车，只学理论又写不出代码 —— 三段式两手都要硬。

## 四、各关卡目标与自测检查点

| 关卡 | 目标 | 过关检查点（能独立做到） |
|---|---|---|
| **第1章 基础** | 数学语言 + Python 工具链 | 用 NumPy 实现矩阵运算与数值梯度；看懂期望/方差/贝叶斯 |
| **Round 1** | 经典 ML + DL 入门 | 从零实现线性/逻辑回归；用 PyTorch 训练一个 MLP/CNN（CPU） |
| **Round 2** | 全考纲 + 现代架构 | 手推注意力；用 GPU 微调预训练模型解决 CV/NLP 任务 |
| **Camp** | 综合实战 | 限时完成一道多域真题，会做误差分析 |
| **Champion** | 国际赛冲刺 | 独立攻克 IOAI/IAIO 历年真题 |

## 小结

- IOAI 考纲 = 基础ML / 深度学习 / CV / NLP 四板块；**未列出即排除**（不考 RNN/RL/时序）。
- 本站把考纲铺在 **四关卡难度主轴**上，同一主题逐关加深。
- 每个知识点用**直觉 → 数学 → 代码**三段式学透。
