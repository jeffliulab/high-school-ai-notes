# 1.3.3 概率与统计

> **难度** ⭐⭐☆☆☆ · **前置**：[1.3.2 微积分与梯度](1-3-2-calculus-gradients.md)

机器学习是在**不确定性**下做决策：数据有噪声、预测有把握程度。概率给了我们度量不确定性的语言，统计给了我们从数据里"反推"规律的工具。很多损失函数（比如交叉熵）其实就是概率公式的伪装。

!!! abstract "学习目标"
    - 理解随机变量、期望、方差，认识伯努利与高斯分布
    - 会用贝叶斯公式做"由果推因"的更新
    - 理解极大似然估计（MLE），知道它如何导出常见损失函数

## 一、直觉：它解决什么问题

模型输出"这张图是猫的概率 0.92"——这个 0.92 就是概率。训练时我们问："什么样的参数，最可能产生我看到的这批数据？"这正是**极大似然**的思路。

## 二、原理与数学推导

**随机变量与期望/方差**：离散变量的期望 $\mathbb{E}[X]=\sum_x x\,p(x)$，方差 $\mathrm{Var}(X)=\mathbb{E}[(X-\mathbb{E}X)^2]$。

**两个核心分布**：

| 分布 | 用途 | 关键量 |
|---|---|---|
| 伯努利 $\mathrm{Bern}(p)$ | 二分类标签（0/1） | $P(X=1)=p$ |
| 高斯 $\mathcal{N}(\mu,\sigma^2)$ | 连续噪声、初始化 | 均值 $\mu$、方差 $\sigma^2$ |

**贝叶斯公式**（由果推因）：

$$P(A\mid B)=\frac{P(B\mid A)\,P(A)}{P(B)}.$$

**极大似然估计 MLE**：给定独立数据 $\{x_i\}$，选参数 $\theta$ 使似然 $\prod_i p(x_i;\theta)$ 最大。取对数把连乘变连加，更好优化：

$$\hat\theta=\arg\max_\theta \sum_i \log p(x_i;\theta).$$

!!! example "例 1：抛硬币的 MLE"
    抛 $n$ 次出现 $k$ 次正面。似然 $L(p)=p^k(1-p)^{n-k}$，对 $\log L$ 求导置零，解得 $\hat p=\dfrac{k}{n}$——**最可能的概率就是观测频率**。最大化对数似然 = 最小化交叉熵，这就是分类损失的来源。

## 三、代码实现

```python
import numpy as np
rng = np.random.default_rng(0)

# 从高斯分布采样，再用样本反估均值/方差（统计的核心动作）
samples = rng.normal(loc=5.0, scale=2.0, size=10000)
print(samples.mean(), samples.var())   # ≈ 5.0, ≈ 4.0

# 抛硬币的 MLE：估计的 p 就是正面频率
flips = rng.binomial(n=1, p=0.3, size=1000)
print("MLE p̂ =", flips.mean())          # ≈ 0.3
```

## 四、常见陷阱与调试

- **概率密度 ≠ 概率**：连续分布里 $p(x)$ 可以大于 1，只有积分才是概率。
- **数值下溢**：很多概率连乘会趋近 0，**永远在对数空间计算**（`np.log`、`logsumexp`）。
- **独立性假设**：朴素贝叶斯假设特征独立，现实常不成立，但仍常奏效——别误以为它"严格正确"。
- **样本太少**：用频率估概率时，$n$ 小则估计方差大，注意平滑（如拉普拉斯平滑）。

## 七、小结与延伸阅读

- 概率度量不确定性；**期望/方差/分布**是基本语言。
- **贝叶斯**做"由果推因"，**MLE**用"最可能产生数据"来选参数。
- 交叉熵损失就是负对数似然——损失函数常是概率的化身。
